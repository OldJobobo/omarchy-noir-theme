# Omarchy Noir Theme

A dark Omarchy theme built around near-black surfaces, parchment foregrounds, worn amber accents, clay reds, moss greens, smoked blues, and muted plum.

## Preview

![Noir dark preview](preview-darl.png)

![Noir preview](preview.png)

## Install

Use the Omarchy theme installer with this repository URL:

```bash
omarchy-theme-install <repo-url>
```

## What's Included

- Omarchy shell, Hyprland, Hyprlock, Walker, Waybar, Mako, SwayOSD, GTK, Chromium, and terminal theme files.
- Terminal palettes for Alacritty, Foot, Ghostty, Kitty, and Warp.
- Editor and app theme ports for Neovim, Zed, btop, and Vencord.
- A Base24 palette export in `noir-base24.yaml`.
- Wallpapers in `backgrounds/`.
